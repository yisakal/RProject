##(extra credit) 10. Lets take a look at the time it took for these alleged encounters to be reported. Compare the encounter date/time to the posted date. What is the mean durations between the two? Hint: maybe take a look here: https://lubridate.tidyverse.org/reference/time_length.html

# measure the delay between when a UFO sighting occurred (date_enc) and when it was posted (date_posted2)

ufo <- ufo %>%
  mutate(

##17. This might display too many items for a concise plot. Can you reduce the count categories and replot?

ufo %>%
